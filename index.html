How it works
1. Share your code above with a friend
2. They enter your code in Add Friend
3. You'll see their request in Requests
4. Accept → they appear in your Friends list
5. Challenge them to a live PVP quiz! ⚔️
⚠️ Friends not working? Fix Firebase Rules
Go to Firebase Console → Firestore → Rules and replace everything with:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read: if request.auth != null; allow write: if request.auth != null; } match /matches/{matchId} { allow read, write: if request.auth != null; } match /feedback/{docId} { allow read, write: if request.auth != null; } } }
Then click Publish. This allows all logged-in users to read/write, which is needed for friends, PVP, and feedback.
